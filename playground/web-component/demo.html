🌟 Sofia Core API

Interactive Web Component Playground

🚀 Quick Start

  • Enter your Sofia Core API key in the input field above the chat
  • Type a message in the text area
  • Press Enter to send (Shift+Enter for new line)
  • Watch Sofia respond in real-time!

📦 Using This Component

This is a framework-agnostic Web Component. Add it to any HTML page:

<!-- Include the script -->
<script src="sofia-playground.js"></script>

<!-- Use the component -->
<sofia-playground></sofia-playground>

Works with React, Vue, Angular, Svelte, or plain HTML. Zero dependencies! 🎉
