# Governance Policies

The Sofia Core SDK governance framework consists of:

1. **[Maintainer Oath](maintainer-oath.md)** - Constitutional commitment to system identity
2. **[Code Review Rubric](code-review-rubric.md)** - Evaluation framework for all PRs
3. **[Issue Triage Protocol](issue-triage-protocol.md)** - Systematic issue handling

Together, these ensure:
- Constitutional integrity (Oath)
- Operational discipline (Rubric, Triage)
- Predictable evolution and safety

Future governance artifacts will include:
- Versioning Policy - Semantic versioning rules
- Deprecation Workflow - Feature lifecycle management
- Security Policy - Vulnerability reporting and handling
