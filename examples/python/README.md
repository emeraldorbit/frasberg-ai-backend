# Sofia Core - Python Example

This example demonstrates using Sofia Core with Python.

## Setup

1. Install dependencies:

```bash
pip install requests
```

2. Set environment variables:

```bash
export SOFIA_URL="https://sdtilgpppwhwtbxlbmik.supabase.co/functions/v1/sofia-core-backend"
export SOFIA_API_KEY="your-api-key"
```

## Run

```bash
python example.py
```

## Examples Included

1. **Basic Chat** - Simple question and response
2. **Streaming Chat** - Real-time streaming responses
3. **Chat with Options** - Using temperature and max_tokens
4. **Sovereign Mode** - Using Sofia-specific extensions
